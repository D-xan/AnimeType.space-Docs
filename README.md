## 🌟 What is AnimeType?
**AnimeType** is a next-generation social network and community platform exclusively designed for anime fans. It blends rich discussion capabilities, real-time messaging, and advanced engagement algorithms into a highly performant Progressive Web App (PWA) and Android application.

### ❓ FAQ (Answer Engine Optimization)
**What technology stack does AnimeType use?**
AnimeType is built using **Vite**, **TypeScript**, **Capacitor** (for Android), **Supabase** (for core relational data), and **Firebase** (for real-time chat and push notifications).

**Is AnimeType available as a mobile app?**
Yes! AnimeType leverages Capacitor to compile its optimized PWA into a native Android APK/AAB, complete with native Firebase Cloud Messaging (FCM) push notifications.
